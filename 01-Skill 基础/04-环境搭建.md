## 🛠️ 安装步骤

### Step 1: 安装 Node.js

**macOS (Homebrew)**:
```bash
brew install node@20
node --version  # 验证安装
```

**Windows**:
```bash
# 下载安装包
https://nodejs.org/
```

**Linux**:
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
```

### Step 2: 安装 OpenClaw

```bash
# 全局安装
npm install -g openclaw

# 验证安装
openclaw --version
```

### Step 3: 配置 OpenClaw

```bash
# 初始化配置
openclaw init

# 查看配置
openclaw config get
```

## 📁 创建第一个 Skill

### Step 1: 创建项目目录

```bash
mkdir my-first-skill
cd my-first-skill
```

### Step 2: 初始化项目

```bash
npm init -y
```

### Step 3: 创建 SKILL.md

```markdown
# 名称
hello-skill

# 描述
一个简单的问候 Skill

# 版本
1.0.0

# 作者
Your Name

# 触发词
- 你好
- hello
- 打招呼

# 工具
## sayHello
- 描述：向用户问好
- 参数:
  - name (string, 可选): 用户名
- 返回：问候语
```

### Step 4: 创建入口文件

```javascript
// index.js

/**
 * 向用户问好
 * @param {Object} params - 参数
 * @param {string} params.name - 用户名
 * @returns {Promise<Object>}
 */
async function sayHello(params) {
  const name = params.name || '朋友';
  return {
    success: true,
    data: {
      message: `你好，${name}！很高兴见到你！`,
    },
  };
}

// 导出工具
module.exports = {
  tools: {
    sayHello,
  },
};
```

### Step 5: 测试 Skill

```bash
# 安装 Skill
openclaw skills install .

# 列出已安装的 Skill
openclaw skills list

# 测试 Skill
openclaw skills test hello-skill
```

## 🧪 测试配置

### Jest 配置

```javascript
// jest.config.js
module.exports = {
  testEnvironment: 'node',
  coverageDirectory: 'coverage',
  collectCoverageFrom: ['src/**/*.js'],
  testMatch: ['**/*.test.js'],
};
```

### 编写测试

```javascript
// __tests__/hello.test.js
const { sayHello } = require('../index');

describe('sayHello', () => {
  it('应该返回问候语', async () => {
    const result = await sayHello({ name: '张三' });
    
    expect(result.success).toBe(true);
    expect(result.data.message).toContain('张三');
  });

  it('应该使用默认名称', async () => {
    const result = await sayHello({});
    
    expect(result.data.message).toContain('朋友');
  });
});
```

### 运行测试

```bash
# 运行所有测试
npm test

# 运行特定测试
npm test -- hello.test.js

# 生成覆盖率报告
npm test -- --coverage
```

## 🐛 调试技巧

### 日志调试

```javascript
// index.js
const logger = {
  info: (msg, data) => console.log(`[INFO] ${msg}`, data),
  error: (msg, error) => console.error(`[ERROR] ${msg}`, error),
};

async function sayHello(params) {
  logger.info('执行 sayHello', params);
  
  try {
    const result = { /* ... */ };
    logger.info('sayHello 成功', result);
    return result;
  } catch (error) {
    logger.error('sayHello 失败', error);
    throw error;
  }
}
```
