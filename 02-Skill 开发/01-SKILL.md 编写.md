## 🔧 工具定义

### 基本格式

```markdown
# 工具
## getWeather
- 描述：查询城市当前天气
- 参数:
  - city (string, 必需): 城市名称
- 返回：天气信息对象

## getForecast  
- 描述：查询天气预报
- 参数:
  - city (string, 必需): 城市名称
  - days (number, 可选): 天数，默认 7 天
- 返回：天气预报列表
```

### 参数说明

```markdown
# 工具
## createUser
- 描述：创建新用户
- 参数:
  - name (string, 必需): 用户名，长度 2-20 字符
  - email (string, 必需): 邮箱地址
  - age (number, 可选): 年龄，18-100
  - tags (array, 可选): 用户标签
- 返回：用户对象
```

## 📝 示例对话

### 简单示例

```markdown
# 示例对话
用户：北京天气怎么样？
AI: 北京今天晴，气温 25°C。
```

### 复杂示例

```markdown
# 示例对话
## 示例 1: 查询当前天气
用户：北京天气怎么样？
AI: 北京今天晴，气温 25°C，空气质量良好。

## 示例 2: 查询预报
用户：上海未来 3 天天气
AI: 上海未来 3 天天气预报：
     今天：晴，25°C
     明天：多云，27°C
     后天：小雨，23°C

## 示例 3: 错误处理
用户：无效城市 123 天气
AI: 抱歉，未找到"无效城市 123"的天气信息。
```

```markdown

## searchFile
- 描述：在指定目录搜索文件
- 参数:
  - directory (string, 必需): 搜索目录
  - pattern (string, 必需): 文件名模式
  - recursive (boolean, 可选): 是否递归，默认 true
- 返回：文件列表

# ❌ 过于简单

- 描述：搜索
- 参数：随便
```
